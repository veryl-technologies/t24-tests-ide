# Veryl T24 Tests Editor IDE

Veryl T24 Tests Editor is IDE for authoring Temenos T24 Scripts and is built on top of Robot Framework RIDE (see https://github.com/robotframework/RIDE/ ). See screenshots at [http://veryl.eu/index.php?option=com_content&view=article&id=124&Itemid=343]

The test cases are plain text filed with RF syntax. See examples at https://github.com/veryl-technologies/t24-tests-library 

The execution engine is https://github.com/veryl-technologies/t24-web-api
